## VPN
国内的小伙伴们很多时候会需要用到VPN或各类加速器。群里大部分人用的是[快连](https://letsvpn.world/)（非广告）VPN，相对质量稳定价格低廉，可以找群内的朋友要邀请码。

## Discord
大家一般语音会用Discord软件，像是海外版的YY语音。Discord有PC和手机客户端，但PC端玩游戏更方便。Discord 国内的朋友用是要VPN的。
